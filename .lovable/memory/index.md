# Project Memory

## Core
"Twilight Deck" theme — softer, lighter futuristic dark mode. Lavender-slate surfaces (oklch ~0.32 0.02 270), warm coral accent (oklch 0.78 0.13 35) — intentionally less masculine than pure amber. Avoid pure black backgrounds. Space Grotesk headings, JetBrains Mono mono, Cormorant Garamond for the brand serif.
Brand name: "Idon Suite". "Idon" = Old English serif (Cormorant Garamond, italic) — old-fashioned but legible. "Suite" = mono uppercase tracked. Joke: archaic word for an AI platform. Use <BrandMark /> from src/components/brand/BrandMark.tsx.
Target audience: busy small biz owners on marketplaces/Shopify. Powerful but approachable.
CSS token name `--amber` is kept for backwards compatibility but now holds the coral accent — do not rename without sweeping the codebase.
