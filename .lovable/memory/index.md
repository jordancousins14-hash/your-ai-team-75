# Project Memory

## Core
Dark "Flight Deck" theme. Amber accent, dark steel bg. Space Grotesk headings, JetBrains Mono mono.
Brand name: "Anlic Suite". "Anlic" = Old English serif (Cormorant Garamond, italic) — old-fashioned but legible. "Suite" = mono uppercase tracked. Joke: archaic word for an AI platform. Use <BrandMark /> from src/components/brand/BrandMark.tsx.
Target audience: busy small biz owners on marketplaces/Shopify. Powerful but approachable.
